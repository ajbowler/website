Graphalyzer

Graphalyzer was my engineering senior design project, a joint project with Workiva. The purpose of Graphalyzer is to transpose large amounts of arbitrary data into an interactive graphical visualization. User interaction with the graph gives more detailed results on the data itself. Graphalyzer also performs graphical analysis, distinguishing important relationships between the data through basic attributes such as color and shape.

SynthePizer

SynthePizer is a musical instrument for the Raspberry Pi. The instrument is played with a Boogie Board eWriter and the UI is all written on a 32x32 LED RGB matrix board. The user selects an instrument to play, and moving up and down the Boogie Board changes the pitch. Everything was written in Python.

This project really gave me newfound appreciation for people who do audio processing for a living. The Raspberry Pi's sound drivers are total garbage, and the Pi has very limited memory to work with, so we had to very carefully work with running two processes at the same time, one for the LED board and one for the audio.

This project was completed start to finish in less than 36 hours as part of the Spring 2016 hackathon HackISU at Iowa State Univerity. I worked on this with Thomas Kearney, Alex Lende, and Nick Montelibano.

Check out the source code!

Check out our live demo!

Voice-A-Sketch

Voice-A-Sketch is a web browser Etch-A-Sketch controlled by your voice! This was a HackISU hackathon project. I learned how to build a UI with Polymer and this was also my first foray into voice recognition tech using the Web Speech API.

Shout out your commands by giving it cardinal directions (North, South, East, West), tell it to "Stop", or have it "Clear" the board.

I worked on this with Thomas Kearney, Shuyang Sun, and Michael Phippen.
